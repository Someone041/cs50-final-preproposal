CS50 Final Project Preproposal
Project Idea
I wish to build a chatbot that can take input data in any form of text and generate quizzes using AI. This means that users can input study materials in any form, and a quiz can be generated automatically to assist in studying.

Motivation
I believe that making a quiz can be a tedious process. However, making a quiz using AI can greatly assist in studying because it can be more efficient and interactive.

Tools & Technologies
Python to be used in the backend
Open API or any other natural processing language to generate questions
Flask or Streamlit to build a simple web interface
SQLite or JSON to store data
Challenges
Ensuring that questions are accurate, meaningful, and flexible enough to be used in any field of study
Ensuring that input data can be handled efficiently
Ensuring that it has a simple interface to use

Next Steps
Experiment with AI question generation, build a simple chatbot interface, and refine output accuracy and quiz variety.
