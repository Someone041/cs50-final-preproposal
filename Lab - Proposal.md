# Proposal

## What will (likely) be the title of your project?

AI Quiz Generator Chatbot

## In just a sentence or two, summarize your project. (E.g., "A website that lets you buy and sell stocks.")

A web-based chatbot that allows users to input study material and automatically generates quizzes using AI to help with studying and review.

## In a paragraph or more, detail your project. What will your software do? What features will it have? How will it be executed?

This project is an AI-based quiz creator that accepts text input from the user in the form of notes, articles, or study material and creates quiz questions in return. The application will allow users to input text material, which will then be passed to an AI model to generate quiz questions of various types.

The application will be created using Python as the backend language, along with an API such as OpenAI to generate quiz questions. A simple interface will also be created using Flask or Streamlit, allowing users to interact with the chatbot, input study material, and generate quizzes instantly. Additionally, the application will allow users to save generated quizzes locally using SQLite or JSON for future use.

Some of the major features of this application will include:

Inputting text material by the user
Using AI to generate quizzes automatically
Showing quizzes in an organized format
Saving generated quizzes for future use
